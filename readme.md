# 📱 StudyFlow Wireframes

## Overview
StudyFlow Wireframes is a UI/UX case study for a Student Study Planner mobile application. This project focuses on understanding user needs and designing low-fidelity wireframes before development.

---

## Problem Statement
Many students struggle to organize assignments, exams, and study schedules. Existing apps are often too complex or lack features tailored to students.

---

## Objectives
- Conduct user research
- Create a user persona
- Design a user flow
- Build low-fidelity wireframes
- Develop a clickable prototype
- Document the design thinking process

---

## Design Thinking Process

### Empathize
Interviewed students to understand their study habits and pain points.

### Define
Students need a simple application to manage tasks and deadlines.

### Ideate
Designed features such as:
- Task Manager
- Deadline Reminders
- Daily Planner
- Progress Tracking

### Prototype
Created low-fidelity wireframes using Figma.

### Test
Collected feedback from peers and improved the navigation flow.
